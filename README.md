# Verify-Public-Key
Verify  a bitcoin public key is valid and lies on the elliptic curve<p>
Example  windows<p>
python39 point.py<p>
Enter public key in hex format: 02CEB6CBBCDBDF5EF7150682150F4CE2C6F4807B349827DCDBDD1F2EFA885A2630<p>
Point is on curve

Linux<p>
python3 point.py<p>
Enter public key in hex format: 02CEB6CBBCDBDF5EF7150682150F4CE2C6F4807B349827DCDBDD1F2EFA885A2630<p>
Point is on curve
